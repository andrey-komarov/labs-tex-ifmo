\section{Цели работы}
\begin{enumerate}
    \item Экспериментальное определение формы эквипотенциальных поверхностей в моделях
плоского и цилиндрического конденсаторов.
    \item Расчет напряженности электростатического поля по найденному распределению	потенциала.
	\item Проверка теоретических предсказаний относительно координатной зависимости
потенциала для обеих моделей.

\end{enumerate}
